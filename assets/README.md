# Assets

Place static branding assets here.

## Suggested files

| Filename        | Description                              | Size        |
|-----------------|------------------------------------------|-------------|
| `logo.png`      | Your logo / initials mark (circular)     | 200×200 px  |
| `banner.png`    | Wide banner for social sharing           | 1280×640 px |
| `favicon.ico`   | Browser favicon                          | 32×32 px    |
| `og-image.png`  | Open Graph preview image                 | 1200×630 px |

## Brand Colors

| Name            | Hex       | Usage                    |
|-----------------|-----------|--------------------------|
| Neon Blue       | `#00d4ff` | Primary accent, glows    |
| Electric Purple | `#a855f7` | Secondary accent          |
| Deep Space      | `#030308` | Background               |
| White           | `#ffffff` | Primary text             |
